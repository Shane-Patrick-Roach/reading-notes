# Reading notes for Navigating the Terminal

## Useful Sources

* [Visual Studio](./https://code.visualstudio.com/docs)

What is a terminal?

* Command line or terminal is a text based interface to the system.

The shell, bash

* This defines how the terminal will behave and looks after running commands for you. You can type **echo** to display a system variable stating your current shell

## Basic Commands **Cheat Sheet**

### Where are we?

* pwd-Print Working Directoy

### Whats in our current location?

* ls-list

### How do I move around?

* cd - change directory

### How do I make a directory?

* mkdir - make a directory (folder)

### How do I obtain information about what type of file a file or directory is?

* file

### Other

* rm - remove file
* touch - create file
* cp - copy file
* mv - move file

## Notes

* Everything is a file.
* Case sensitive
* Quotes or Escape Characters are needed to enter a program.
